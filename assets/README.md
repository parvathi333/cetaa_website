# CETAA Website Image Assets

To complete this prototype, please place your custom images in this `assets/` folder. The website uses standard CSS fallbacks (gradients + icons), so it will work immediately, but adding these images will make it look completely live!

### Required Image Filenames:

1. **`campus_hero.jpg`**
   - **Where it is used**: Home page Hero Background
   - **Recommended Dimensions**: `1920x1080` (landscape, high quality)
   - **Visual style**: Panoramic or wide-angle shot of the iconic College of Engineering Trivandrum (CET) campus.

2. **`about_heritage.jpg`**
   - **Where it is used**: Home page "Our Heritage & Story" section
   - **Recommended Dimensions**: `800x600` (landscape/ratio 4:3)
   - **Visual style**: Traditional buildings, older campus photos, or historical founding members.

3. **`news_upsc.jpg`**
   - **Where it is used**: Home page "Wall of Pride" - UPSC section
   - **Recommended Dimensions**: `600x400`
   - **Visual style**: Civil services topper graduates or academy symbols.

4. **`news_innovation.jpg`**
   - **Where it is used**: Home page "Wall of Pride" - Innovation section
   - **Recommended Dimensions**: `600x400`
   - **Visual style**: Student laboratory work, robotics, or innovation projects.

5. **`news_lalithambika.jpg`**
   - **Where it is used**: Home page "Wall of Pride" - Dr. Lalithambika spotlight
   - **Recommended Dimensions**: `600x400`
   - **Visual style**: Dr. V. R. Lalithambika or space mission elements.

6. **`hall_facade.jpg`**
   - **Where it is used**: Renovation page Hero sidebar
   - **Recommended Dimensions**: `800x600`
   - **Visual style**: External facade or entrance portal of the Diamond Jubilee Hall.

7. **`hall_before.jpg`**
   - **Where it is used**: Renovation page comparison slider (Left side / "Before")
   - **Recommended Dimensions**: `1200x840` (Maintain same size as `hall_after.jpg`)
   - **Visual style**: Interior view of the traditional Hall (traditional wood, seating, fans).

8. **`hall_after.jpg`**
   - **Where it is used**: Renovation page comparison slider (Right side / "After")
   - **Recommended Dimensions**: `1200x840` (Maintain same size as `hall_before.jpg`)
   - **Visual style**: 3D design rendering of the proposed renovated air-conditioned auditorium.

---
*Note: You can use `.png` or `.jpg` formats. If you use `.png`, you can simply update the extensions inside `index.html` and `renovation.html`, or keep them as `.jpg` since modern browsers will handle some formats automatically or you can match the file extensions in the HTML code.*
